# Examples

This directory will contain example payloads, scenarios, and usage patterns that illustrate the specification in practice.

## Purpose

Examples serve as:

- Non-normative illustrations of spec concepts
- Reference material for implementers
- Test case inspiration for conformance work

## Status

Examples will be created after the specification documents have progressed beyond the outline stage. Premature examples risk anchoring designs to specific formats before the architecture is stabilized.

## Contributing Examples

When contributing examples, please:

- Reference the spec document and profile the example illustrates
- Clearly mark examples as non-normative
- Include both valid and invalid examples where instructive
